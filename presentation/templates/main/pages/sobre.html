Sobre o Cidapro

O Cidapro é um projeto desenvolvido para a disciplina de Desenvolvimento Web, ministrada pelo Professor Doutor Ricardo Maroquio, para os alunos do sétimo período do curso de Bacharelado em Sistemas de Informação do Instituto Federal do Espírito Santo, campus Cachoeiro de Itapemirim. Este projeto consiste em uma aplicação web que visa facilitar a obtenção de cidadania estrangeira para brasileiros, idealizado pelos alunos Igor Almeida da Silva e Júlia Borges Santos. Utilizando as melhores práticas de desenvolvimento aprendidas com o professor, assim como estudos próprios dos alunos, o Cidapro combina conhecimento acadêmico e prático para oferecer uma solução eficiente e acessível para aqueles que buscam adquirir cidadania estrangeira.
